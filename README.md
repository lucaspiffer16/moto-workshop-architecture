This repository contains the **architecture and technical documentation** for the Moto Workshop management system.

The system is a **Windows-first, offline-first** desktop application designed for small motorcycle workshops, focusing on work orders, parts stock, and daily operations.

This repository documents **design decisions, system structure, and database schemas**.
Application source code lives in a separate repository.

## Key Concepts
- .NET MAUI
- MVVM with feature-based architecture
- SQLite (local database)
- Offline-first design
